CONVOLUTIONAL NEURAL NETWORK (CNN) METHOD OPTIMIZATION WITH DROPOUT, WEIGHT REGULARIZER L1, AND EARLY STOPPING FOR CLASSIFYING MELANOMA AND NEVUS SKIN CANCER

The idea of this final project is to reduce overfitting in CNN method when classifying skin cancer melonoma or nevus with adding combination of dropout, weight regularizer L1, and early stoping. The output is a CNN model that wrapped with web based application that was developed using Python with Django Framework.
